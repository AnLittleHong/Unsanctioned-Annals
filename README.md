# 互联网野史

> 以民间视角，记录当下中国互联网上真实发生的事件。不为流量，不站立场，只求把事实讲清楚。

---

## 这是什么

《互联网野史》是一个个人记录项目。当互联网热点一个接一个地爆发、发酵、消失，大多数讨论在七天后就被彻底遗忘——这个项目的目标是把那些值得被记住的事件，用经得起时间检验的方式记录下来。

每一篇记录都遵循同一套方法：

1. **事实层** — 发生了什么？证据是什么？所有事实按 S/A/B/C/D 五级标注可信度，不确定的明确标注"待核实"。
2. **舆论层** — 人们怎么说？情绪分布是什么？不同平台、不同群体的反应分别记录。
3. **因果层** — 为什么会发生？深层原因是什么？先收集各方解读，再做出记录者独立判断。
4. **立场层** — 记录者怎么评价？明确声明立场，事实与评价严格分离。
5. **叙事层** — 综合以上，书写最终正文。

核心原则只有一条：**事实与评价分离。不确定的事，就说不确定。**

## 阅读说明

## 关于记录者

一个普通的互联网观察者。不属于任何机构，不服务于任何平台。记录只是因为觉得这些事情值得被记住。

如果你对某篇记录的准确性有疑问，欢迎指出。如果你对同一事件有不同的视角和补充，同样欢迎。所有事实层面的讨论都是开放的——但请带上你的证据。
